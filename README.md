# AlphaTauro Panama

Proyecto base de React + Vite listo para desplegar en Vercel.

## Scripts

- `npm run dev`: servidor de desarrollo
- `npm run build`: build de producción
- `npm run preview`: vista previa local del build

## Despliegue en Vercel

1. Sube este proyecto a un repositorio de GitHub, GitLab o Bitbucket.
2. Importa el repositorio en Vercel.
3. Vercel detectará automáticamente Vite.
4. Usa `npm run build` como comando de build si Vercel te lo pide.
